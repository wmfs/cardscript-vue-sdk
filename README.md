# cardscript-vue-sdk

> An SDK for using Cardscript with Vue/Vuetify

## <a name="install"></a>Install
```bash
$ npm install cardscript-vue-sdk --save
```

## <a name="test"></a>Testing

```bash
$ npm test
```
